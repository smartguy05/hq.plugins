# HQ.Logging.FileLogger

## Overview

**HQ.Logging.FileLogger** is a plugin for the [HQ](https://github.com/smartguy05/hq.plugins) ecosystem. It provides robust and configurable file-based logging capabilities for orchestrated AI workflows running in .NET environments. This plugin allows you to persist logs to disk, enabling easier troubleshooting, auditing, and analysis of orchestrator and plugin activities.

## Features

- Write logs to file with customizable formats
- Supports various logging levels (Debug, Info, Warning, Error, Fatal)
- Configurable log file location and rotation
- Simple integration with HQ plugin system

## Installation

Add the complied plugin into the Plugin folder and the config file (HQ.Logging.FileLogger) in the Configs folder of HQ.
